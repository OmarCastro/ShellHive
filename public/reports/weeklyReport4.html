Week 09/02/2012 - 15/03/2012

Tasks done

I had less available time than expected in this week, as such, next list shows the implementation tasks that has been done:

  • Parse of redirection to visual representation
  • Added file component (incomplete)
  • Improved Interface
    • Allows the removal of components
    • Implemented a feature that creates a component by connecting a port to an free space
    • The same feature allows to create a component by writing a text command with options (e.g. "cat -A")
    • improved menus locations
  • Bug fixes on some commands ( specially "head", "tail" and "compress") and on the parser
  • More Tests
  • Implemented creation of Macros
  • Continue the implementation of Macros
  • Created a Demo page

Note

While implementing the macros, a problem has surfaced when dealing with "macros inside macros", which is that exists a possibility of creating a cycle (e.g. creating a puting the same macro inside it)

As such, I decided to restict that the macros should not have "submacros" until I implment an algorithm to find cycles (and I believe it can be a low priority task).

Interaction
