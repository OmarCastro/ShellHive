List of tasks done

In this week I finished revising the technical report, then I started the implementation of the Unix commands parser. The next list shows the implementation tasks that has been done this week:

  • Unix command parser to intermediate abstract syntax tree (AST)
  • Parse of the "cat" command to visual representation
  • Parse of the "grep" command to visual representation
  • Parse of the "bzip2" command to visual representation
  • Parse of the "bzcat" command to visual representation
  • Parse of the "bunzip2" command to visual representation
  • Parse of the "compress" command to visual representation
  • Parse of the "ls" command to visual representation (incomplete)
  • Parse of visual representation of "cat" command to shell code
